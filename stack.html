Stack (Tumpukan)

Stack adalah struktur data yang menggunakan prinsip LIFO (Last In First Out). Artinya: data terakhir yang masuk akan keluar pertama.

Ilustrasi

Implementasi Python


stack = []

# Push (menambah data)
stack.append(10)
stack.append(20)
stack.append(30)

print("Stack:", stack)

# Pop (menghapus data terakhir)
stack.pop()

print("Setelah pop:", stack)

Penjelasan :

Misal tumpukan piring: Masuk: 10 → 20 → 30 Keluar: 30 dulu
