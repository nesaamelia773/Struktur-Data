Prefix to Infix

Prefix : +AB

Ilustrasi

Implementasi Python


def prefix_to_infix(expression):
    stack = []

    for char in reversed(expression):
        if char.isalnum():
            stack.append(char)
        else:
            a = stack.pop()
            b = stack.pop()
            stack.append(f"({a}{char}{b})")

    return stack[0]

print(prefix_to_infix("+AB")

Penjelasan :

Baca dari kanan Jika operand → masuk stack Jika operator → ambil 2 operand Hasil: Infix = (A+B)
