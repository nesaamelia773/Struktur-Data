Postfix to Infix

Postfix : AB+

Implementasi Python


def postfix_to_infix(expression):
    stack = []

    for char in expression:
        if char.isalnum():
            stack.append(char)
        else:
            b = stack.pop()
            a = stack.pop()
            stack.append(f"({a}{char}{b})")

    return stack[0]

print(postfix_to_infix("AB+"))

Penjelasan :

Baca dari kanan Jika operand → masuk stack Jika operator → ambil 2 operand Hasil: Infix = (A+B)
