Evaluasi Postfix

Postfix : 23*5+

Ilustrasi

Implementasi Python


def eval_postfix(expression):
    stack = []

    for char in expression:
        if char.isdigit():
            stack.append(int(char))
        else:
            b = stack.pop()
            a = stack.pop()

            if char == '+':
                stack.append(a + b)
            elif char == '-':
                stack.append(a - b)
            elif char == '*':
                stack.append(a * b)
            elif char == '/':
                stack.append(a // b)

    return stack[0]

print(eval_postfix("23*5+"))  # (2*3)+5

Penjelasan :

isdigit() → masuk stack operator: b = stack.pop() a = stack.pop() ➜ ambil 2 angka terakhir hitung → masukkan lagi ke stack Hasil akhir: 11
